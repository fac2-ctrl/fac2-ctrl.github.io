Hi everyone! Here's my Help, I need somebody project. It plays morse code for my name with unconnected, and SOS when connected. A tip I'd give for completing this project is while testing set the speed fast and the frequency to a tolerable tone. It makes it much easier to hear the difference between the diferent modes.
