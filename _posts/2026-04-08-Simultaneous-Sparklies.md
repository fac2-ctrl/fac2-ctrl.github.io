Hi everyone! Here's my simultaneous sparklies project! This project was really just about combining the separate sparklies. Because I used a state based aproach, it was easy to put them together to make a full simulation of a night in the Great Smoky Mountains. A tip I'd give to someone doing this projest is that it really is that easy to put together code when you use a state based aproach, just make sure you only have one delay.
